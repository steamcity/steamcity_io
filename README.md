```bash
npm install

```

## Navigation

- `#/experiments` - Liste des experiences
- `#/experiments/:id` - Detail une experience
- `#/sensors` - Vue des capteurs agreges
- `#/map` - Carte interactive
- `#/charts` - Analyse graphique des donnees
